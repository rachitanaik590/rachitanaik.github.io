## ğŸ› ï¸ Setup & Deployment

### Local Development

1. Clone this repository:
   ```bash
   git clone https://github.com/rachitanaik98/rachitanaik98.github.io.git
   cd rachitanaik98.github.io
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (if you have http-server installed)
   npx http-server
   ```

3. Visit `http://localhost:8000` in your browser

### Deploy to GitHub Pages

1. Create a new repository on GitHub named `rachitanaik98.github.io` (replace with your GitHub username)

2. Initialize git and push to GitHub:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Personal website"
   git branch -M main
   git remote add origin https://github.com/rachitanaik98/rachitanaik98.github.io.git
   git push -u origin main
   ```

3. Enable GitHub Pages:
   - Go to your repository settings
   - Navigate to "Pages" in the left sidebar
   - Under "Source", select "main" branch
   - Click "Save"

4. Your website will be live at `https://rachitanaik98.github.io` (may take a few minutes)
